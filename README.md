# Image-To-Ascii
## Simple web app that converts images into ascii art 

### Live ----> [Image-to-ascii](https://nitch193.github.io/Image-to-Ascii/)

- Upload any image(webp/png/jpg/jpeg)
- ✨Magic ✨
- It does not store your uploaded image on any database.
